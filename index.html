SpRestLib

SharePoint REST Library: JavaScript Library for SP2013+ Web Services

SharePoint REST Library that simplifies AJAX interaction to a few lines of code.
Easily read items, perform CRUD operations, gather user information and populate form elements.

Features

Modern API

Implmements the current SharePoint 2013 API / OData v3

Easy Async

Utilizes the new ES6 Promise architecture to enable chaining of asynchronous operations

Simple

Most REST interaction can be done in 1-2 lines of code

Form Population

Avoid code altogether and just use data binding syntax to populate form elements

Pure Javascript

SpRestLib only requires the jQuery Library and nothing else

Robust

Handles errors and monitors the SharePoint authentication token

Examples

SpRestLib does the heavy-lifting for you!

Get List Items

sprLib.list('Employees')
.getItems({
    listCols: ['Id', 'Name', 'Hire_x0020_Date'],
    queryFilter: 'Id eq 99',
    queryOrderby: 'Name'
})
.then(function(arrData){ console.table(arrData); })
.catch(function(errMsg){ console.error(errMsg);  });

User Information

sprLib.user().info()
.then(function(objSpUser){
    console.log( "User Id....... " + objSpUser.Id );
    console.log( "User Title.... " + objSpUser.Title );
    console.log( "User Email.... " + objSpUser.Email );
    console.log( "USer LoginName " + objSpUser.LoginName );
})
.catch(function(errMsg){ console.error(errMsg);  });

REST Calls

sprLib.rest({
    restUrl:      '/sites/dev/_api/web/sitegroups',
    queryCols:    ['Title', 'LoginName'],
    queryFilter:  'AllowMembersEditMembership eq false',
    queryOrderby: 'Title',
    queryLimit:   10
})
.then(function(arrayResults){
    console.table(arrayResults);
});

CRUD Operations

var item = { Name:'Marty McFly', HireDate:new Date() };

Promise.resolve()
.then(function(){
    return sprLib.list('Employees').create(item);
})
.then(function(item){
    return sprLib.list('Employees').update(item);
})
.then(function(item){
    return sprLib.list('Employees').delete(item);
})
.then(function(item){
    console.log('Ran entire CRUD chain!');
});

Form Population

<body>
    <select data-sprlib='{ "list":"Employees", "value":"Id", "text":"Name" }'></select>

    <input type="text" data-sprlib='{ "list":"Departments", "value":"Title" }' placeholder="Departments.Title"></input>

    <span data-sprlib='{ "list":"Employees", "value":"Name", "filter":{"col":"Name", "op":"eq", "val":"Brent Ely"} }'></span>

    <table data-sprlib='{ "foreach": {"list":"Employees", "cols":["Name","Utilization_x0020_Pct"] } }'></tbody>
</body>

Installation

Client-Side

  • SpRestLib utilizes the jQuery library - include it before sprestlib. That's it!
  • NOTE: IE11 support requires you include a Promises polyfill as well (one is included in the 'libs' folder)
  <script lang="javascript" src="https://code.jquery.com/jquery-3.1.1.slim.min.js"></script>
  <script lang="javascript" src="https://yourhost.com/subsite/SiteAssets/js/sprestlib.js"></script>

Node.js

  • SpRestLib NPM Homepage
  npm install sprestlib
  var sprLib = require("sprestlib");
